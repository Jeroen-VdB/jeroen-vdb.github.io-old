Azure Integration Developer

April 2020 - August 2020

Inetum-Realdolmen @ Hospital Logistics

The company integrated with their customers using BizTalk. Due to the ending lifecycle of BizTalk they wanted to move their solutions to the cloud. Together with a colleague (Integration Architect) we came up with a solution to replace the current BizTalk logic with a new dynamic setup in Azure. We used Azure DevOps to automatically deploy our resources to Azure, including our configuration. With the use of an Integration Account, Logic Apps, Azure Functions, ServiceBus, and a SQL database hosted in Azure we were able to fully mimic the old solution. Furthermore, our client can now easily expand the configuration to onboard new customers with the least amount of effort.
